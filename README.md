# Jogo da Velha - TicTacToe

Projeto realizado para entrevista técnica na empresa transobra.

## Tecnologias

- Next.js
- React
- TypeScript
- TailwindCSS
- Lucide-React
- Recharts

## Pré-requisitos

- Node.js >= 18
- npm ou yarn

## Instalação

1. Clone o repositório:
git clone https://github.com/RaxasDev/tic-tac-toe-app.git

2. Na pasta raíz do projeto, insira o comando `npm i`

3. Na sequência dê o comando `npm run dev`
